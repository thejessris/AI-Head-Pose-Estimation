# AI-Head-Pose-Estimation

Real-time head orientation estimation using OpenCV,
Dlib facial landmarks and Flask.

## Features

✔ Yaw Detection

✔ Pitch Detection

✔ Roll Detection

✔ Real-time Webcam Stream

✔ Interview Monitoring Support

✔ Side Profile Handling

✔ Stable Angle Estimation

## Run

pip install -r requirements.txt

python app.py

Open:

http://localhost:5000
